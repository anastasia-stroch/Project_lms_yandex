Мой проект будет скорее игрой, чем просто приложением. В нём каждый раз будут рандомно составляться рабочие цепочки из логических элементов: AND, OR, NAND, NOR, - цепочки будут видны пользователю и ему нужно будет вводить в поле для ввода какой результат будет на выходе каждой цепочки (у цепочек может быть до двух выходов одновременно), далее система будет проверять ответ и приступать к следующей цепочке, если эта пройдена.
